# List of Browsers and Descriptions

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic HTML webpage that presents a list of major web browsers along with short descriptions of each one.

It was created to practise structuring information clearly using semantic HTML elements, especially the description list format.

## Features

- Main page heading
- Description list of major web browsers
- Short explanatory text for each browser
- Clean and simple HTML structure

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using headings to structure a webpage
- Using the `dl`, `dt`, and `dd` elements
- Presenting information in a clear and readable format
- Building a simple informational webpage
